Portafolio (Proyectos Destacados)

Glove

IONIC 4 / Angular

Fecha: 28/05/2020

El proyecto escolar Glove fue resultado de 4 meses de trabajo, es una aplicación móvil que tiene como objetivo principal ayudar al usuario a mejorar la comprensión del código morse mediante entrenamiento (para ayudar a personas con alguna discapacidad del habla o auditiva), con el cual podrá desarrollar nuevas habilidades y reconocer los patrones del código para formar una oración. También se desarrolló un guante con una pantalla y un botón para que la persona se comunique ya una vez aprendido el código morse. ​
